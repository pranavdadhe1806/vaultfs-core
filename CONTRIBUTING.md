# Contributing to VaultFS

## Setup
1. Fork the repo
2. Clone your fork
3. Set up auth-server locally (see auth-server/README.md)
4. Compile Java and test

## Structure
- Java changes → src/
- Auth server changes → auth-server/
- CLI launcher changes → bin/

## Pull Request
- One feature per PR
- Test on Windows + Mac/Linux if possible
- Bump version in version.txt and package.json
